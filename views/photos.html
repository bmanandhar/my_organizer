Projects that use public API's:

  • Weather App

    It uses API data from the Open Weather Map website. The location has been derived directly as geo-coordinate of the place client-computer accesses IPS that provides internet from. User can check weather condition with an option to get temperature in fahrenheit and as well as celcius using toggle method (https://openweathermap.org/ and https://ipinfo.io).

  • Random Quote Generator

    It uses the API data from the source that generates quotes delivered by different famous persons in the world using click button. User can post these quotes to facebook or twitter (https://forismatic.com/en/).
